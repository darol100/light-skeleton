Documentation

Download
  • Light Skeleton can be download at project page at Drupal.org
  • This project can also be download via drush by running this drush command. 
    drush dl light_skeleton
Installation
  • Install the Light Skeleton in sites/all/themes or a similar sites/*/themes folder.
  • Ensure that jQuery has been configured to meet Light Skeleton minimum version requirement, which is jQuery 1.7.
Configuration
  • Menu: Once Light Skeleton is been "Enable and set default", the "Main menu" block needs to be move to the "Main menu" region. This can be done block configuration (admin/structure/block).
